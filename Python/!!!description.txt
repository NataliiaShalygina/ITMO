Приложение "Математический тренажёр"
Написано на версии Python 3.14. Использует связь с базой данных на SQL Server.

Приложение создано как помощник репетитору для проведения онлайн-занятий по математике. С помощью данного приложения можно генерировать различные задачи для отработки навыков счёта, а также подгружать из Базы Данных задачи из различных источников (учебников и книг) или по выбранным темам. В будущем планируется добавление доски для демонстрации задач на построение (геометрия).  

Необходимо установить: 
1. Библиотеки для Python: pillow, pip, pyodbs( файл list.txt)
2. Драйвер для связи с БД (Driver 18 for SQL Server) : https://learn.microsoft.com/en-us/sql/connect/odbc/download-odbc-driver-for-sql-server?view=sql-server-ver17

Структура приложения.

Приложение состоит из четырёх страниц:
1. Стартовая (main), предлагает пользователю выбор соответствующего раздела.
2. Арифметика, включающая в себя тренажёры для счёта.
3. Алгебра, где можно найти уравнения и неравенства.
4. Геометрия, предлагающая задачи по разным темам (в будущем планируется добавление доски).

	
	На первой странице три иконки для вывода информации, под каждой из которых находятся кнопки "Сгенерировать". В первой иконке генерируется пример для тренировки устного счёта, во второй - пример для отработка действий с обыкновенными дробями, в третьей - для смешанного вычисления. Данные опции позволяют не искать примеры для отработки навыков счёта, а сгенерировать их одним нажатием кнопки. Для генерации примеров и отображения дробей используются методы встроенной библиотеки Math. 

	На второй странице три иконки для вывода, две из которых позволяют или вывести задачу из БД, или сгенерировать свою. Представлены по трём блокам - линейные уравнения, квадратные уравнения, неравенства. Изначально окно отображает значение пустой переменной, которая обновляется в зависимости от выбранного метода - "Сгенерировать" или "Найти". При выборе первой кнопки переменной присваивается строка со сгенерированным заданием. При выборе второй через инструмент класса Навигатор, осуществляется запрос в базу данных (в зависимости от темы), обновление виджета и переключение в случае, если задач больше одной.

	На третьей странице представлена иконка для вывода текста задачи, а также два алгоритма для поиска. Они отличаются тем, что используют разные методы поиска задач в БД. Первый метод предполагает выбор темы и типа задачи, для визуализации выбора создаются выпадающие списки. Первый список берёт значения непосредственно из БД (во избежание ошибок), второй задаётся вручную, так как содержит всего три параметра. При выборе темы и типа задачи выбранные значения с помощью специальных обёрток передаются в соответствующие методы класса Навигатора, если ничего не выбрано, передаются значения по умолчанию. Из метода поиска происходит вызов метода обновления виджета. Второй метод предполагает использование списка и передачи ручного значения. Чтобы в качестве значения передавалось только целое число, используется метод для отслеживания переменной. Непосредственно перед передачей значений переменных методу происходит конвертация в число.


Файлы программы:
1. Файл main. py для запуска приложения и отображения главного окна.
2. Файлы arithmetic_window.py, algebra_window.py, geometry_window.py - содержат классы трёх окон.
3. Файл base_window.py содержит описания классов основных элементов отображения: окна, иконки (для выбора), иконки (для отображения), выпадающие списки, кнопки.
4. Файл bd.py используется для описания класса работы с БД. Описано формирование строки подключения, метод создания объекта, также три метода для формирования и осуществления запроса и два метода для выгрузки значений для списка.
5. Файл taskNavigator.py описывает класс Навигатора. В данном классе представлены методы по: сохранению переданных значений (из окон алгебры и геометрии), поиска по выбранным параметрам (три метода), внутри которых вызывается обновление виджета информационной иконки, а также происходит настройка кнопки "Следующая".
6. Картинки для иконок (mapp/...)
7. Файл с описанием используемых библиотек (list.txt)
