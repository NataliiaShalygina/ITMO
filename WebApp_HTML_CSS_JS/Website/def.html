Основные определяемые понятия

Отрезок - часть прямой, ограниченная двумя точками.

Луч - направленный отрезок.

Угол - два луча, исходящие из одной точки.

Пересекающиеся прямые - это прямые, имеющие общую точку.

Параллельные прямые - это прямые, не имеющие общих точек.

Перпендикулярные прямые - это прямые, образующие при пересечении угол в 90 градусов.
