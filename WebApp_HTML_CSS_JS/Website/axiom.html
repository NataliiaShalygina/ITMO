Аксиомы геометрии

1. Через любые две точки проходит прямая, и притом только одна.

2. Из трёх точек на прямой одна и только одна лежит между двумя другими.

3. На любом луче от его начала можно отложить отрезок, равный данному, и притом только один.

4. От любого луча в заданную сторону можно отложить угол, равный данному неразвёрнутому углу, и притом только один.

5. Через точку, не лежащую на данной прямой, проходит только одна прямая, параллельная данной.

Пятый постулат называют евклидовым, именно вместе с ним другие постулаты образуют евликодову геометрию. О других видах геометрии можно прочитать здесь.
