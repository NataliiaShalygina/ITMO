Теоретический базис

Основные понятия

В геометрии есть определяемые и неопределяемые понятия, на основе которых строятся основные элементы и отношения в геометрии.

Аксиомы евклидовой геометрии

Аксиомы - это утверждения, принимаемые без доказательств на основе эмпирического опыта. Именно аксиомы описывают отношения между основными понятиями и являются базисом для построения теории.

Неевклидовы геометрии

В зависимости от набора аксиом геометрии можно разделить на нвклидовы и неевклидовы.

Основные геометрические фигуры

Треугольники

Треугольник - геометрическая фигура, состоящая из трёх точек плоскости, не лежащих на одной прямой, и трёх отрезков, попарно соединяющих эти точки.

Четырёхугольники

Четырёхугольник - геометрическая фигура, состоящая из четырёх точек, никакие три из которых не лежат на одной прямой, и четырёх отрезков, последовательно соединяющих эти точки без самопересечений.

Окружности

Окружность - геометрическая фигура, состоящая из всех точек плоскости, которые находятся на заданном расстоянии от данной точки.
